SwampHacks 2016

SwampHacks is a 24-hour hackathon (software development marathon) at the University of Florida that is intended to bring makers of all experience levels together for a weekend to create some amazing things and learn new skills. It was my first hackathon and we were the only all-female team that competed. The project we submitted used Clarafai's api in order to pull the most prominent color from a photo and convert the color's hexadecimal code into binary in order to teach children about binary in a fun and interactive manner.
